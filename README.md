# 🍔 Food Ordering Website

A full-stack Food Ordering Website built using the MERN Stack with AI-powered features for an enhanced user experience.

## 🚀 Features

* User Authentication (JWT)
* Restaurant Listing
* Food Menu Management
* Add to Cart
* Order Placement
* Restaurant Reviews & Ratings
* AI Food Description Generator
* AI Review Analyzer
* Admin Dashboard
* Responsive UI

## 🛠️ Tech Stack

### Frontend

* React.js
* Redux Toolkit
* HTML
* CSS
* JavaScript

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose

### AI & Integrations

* OpenRouter AI
* Cloudinary
* Stripe

## 📂 Project Structure

```text
FoodProject/
├── frontend/
├── backend/
```

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/MansviJiwane/FoodProject.git
```

Install dependencies:

```bash
cd backend
npm install

cd ../frontend
npm install
```

Run the backend:

```bash
cd backend
npm run dev
```

Run the frontend:

```bash
cd frontend
npm run dev
```
